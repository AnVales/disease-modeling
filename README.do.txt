======= Disease modeling =======

A basic introduction to disease modeling.

===== Content =====

 * `doc/pub`: published slides, see `doc/web/index.html` for an overview
   of links to the various compiled slide formats
 * `doc/src`: source code for slides
 * `doc/src/src`: source code for the computational examples in the talk
 * `doc/web`: GitHub-generated web pages (entry point for this repo on the web)
